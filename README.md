# Nodemon Demo

Just demonstrating Nodemon for a YouTube Video

[![IMAGE ALT TEXT HERE](https://img.youtube.com/vi/JewzZROAho8/0.jpg)](https://www.youtube.com/watch?v=JewzZROAho8)

## Installation

for `yarn` users

```bash
yarn add nodemon --dev
```

for `npm` users

```bash
npm i -D nodemon
```

## Running It

```bash
nodemon
```
